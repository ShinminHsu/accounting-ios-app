## 1. 帳戶管理

### 支援的帳戶類型

App 支援以下五種帳戶：

- **現金**
- **銀行帳戶**
- **電子支付**（如 LINE Pay、街口）
- **信用卡**（顯示未繳金額，正數 = 你欠銀行的錢）
- **投資**（僅追蹤餘額，不計算損益）

帳戶分為兩大類：**資產**（現金、銀行、電支、投資）和**負債**（信用卡）。

### 資產頁面總覽

- **頂端卡片**：顯示淨資產總額，正值顯示綠色、負值顯示紅色。點眼睛圖示可隱藏/顯示所有金額。
- **2×2 摘要格**：
  - 可支配：非信用卡帳戶加總
  - 負債：信用卡餘額加總
  - 借出：未收回的代付總額
  - 借入：尚欠他人的總額
- 帳戶依類型分組，每組可展開/收合。
- 信用卡帳戶額外顯示剩餘額度與下次繳款日/金額。

### 新增帳戶

1. 在資產頁點選「新增帳戶」。
2. 填寫帳戶名稱、類型、初始餘額、幣別（支援 TWD、USD、JPY、EUR、HKD、CNY 等）。
3. 若選擇信用卡，還需填入：帳單結帳日、繳款截止日、自動扣款銀行帳戶（選填）。

### 外幣匯率設定

App **不自動抓取即時匯率**，需手動設定。在設定中輸入每種幣別對 TWD 的匯率，例如 1 USD = 32.5 TWD。設定後所有外幣帳戶餘額會自動換算成 TWD 顯示在淨資產中。若某幣別未設定匯率，該帳戶餘額不計入淨資產並顯示警示。

### 刪除帳戶

只有「無任何交易且餘額為零」的帳戶才可刪除。

## 2. 記帳（交易記錄）

### 新增一筆支出

點選主頁的「+」按鈕開啟記帳表單，填寫：

| 欄位 | 說明 |
|------|------|
| 金額 | 必填，必須大於 0 |
| 日期 | 必填，預設為今天 |
| 類別 | 必填（父類別，或再選子類別） |
| 帳戶 | 必填（選擇付款帳戶） |
| 帳本 | 選填（預設為個人帳本，或選共享帳本） |
| 付款方式 | 三種模式（見下方） |
| 代付對象 | 非「自己付」時出現 |
| 專案 | 選填 |
| 備註 | 選填 |

### 三種付款方式

**1. 自己付（預設）**
正常支出，從選定帳戶扣除金額，計入預算。

**2. 別人付（paid-by-other）**
別人幫你墊付：
- 系統建立一筆支出記錄（計入預算，但不扣帳戶餘額）
- 同時在「借還款追蹤」建立一筆你欠對方的**負債記錄**
- 代付對象可從好友清單選取，或直接輸入姓名（如「室友小明」）

**3. 幫人付（paid-for-other）**
你幫別人墊付：
- 系統建立一筆現金流出（從帳戶扣款），**不計入預算**
- 同時在「借還款追蹤」建立對方欠你的**應收記錄**

### 日曆檢視

主頁以日曆呈現，每天顯示當日總支出。點選任一天可展開查看當日所有交易（依時間倒序）。

### 搜尋交易

可全文搜尋交易的備註、金額、類別等欄位。

### 編輯 / 刪除交易

- 可修改所有欄位（若該筆已連結借還款記錄，付款方式不可更改）。
- 刪除交易時，連結的借還款記錄也會一併刪除。

## 3. 類別管理

### 兩層類別結構

類別分為**父類別**和**子類別**兩層。記錄交易時可選父類別或更細的子類別。

### 預設類別

首次啟動 App 自動建立以下類別：

- 餐飲（早餐、午餐、晚餐、咖啡、飲料）
- 交通（大眾運輸、計程車/共乘、油費、停車費）
- 購物（服飾、電子產品、日用品）
- 娛樂（電影、遊戲、訂閱服務）
- 醫療（藥局、診所、健身房）
- 居住（房租、水電、維修）
- 旅遊、教育、其他

### 自訂類別

- 可新增父類別或在任一父類別下新增子類別
- 每個類別可設定名稱與圖示（從圖示選擇器中選取視覺化圖示）
- 名稱不可在同一層重複

### 刪除類別

只有**無任何交易**的類別才可刪除。若已有交易，系統會提示先重新分類。

## 4. 借還款追蹤

入口：「更多」分頁 → **借還款追蹤**

### 功能概覽

顯示每個對象的**淨餘額**：
- 正數 = 對方欠你
- 負數 = 你欠對方

例：你欠男友 200 TWD + 300 TWD，男友欠你 100 TWD → 淨額 -400 TWD（你欠男友 400）

### 未結清記錄

每位對象下列出所有未結清的個別借款/代付記錄。

### 記錄還款

1. 進入對象的借還款清單，選擇要還的一筆記錄。
2. 填入還款金額（支援部分還款）、日期、來源/目標帳戶。
3. 全額還清後該記錄標記為「已結清」。

### 對象識別

- 已加為好友的聯絡人：顯示好友名稱
- 未加好友、直接輸入名字的對象：以輸入的文字名稱（如「室友小明」）分組顯示

### 標記爭議

可將任一借款記錄標記為「有爭議」，系統會以特殊樣式顯示，但仍納入餘額計算。

## 5. 信用卡對帳

入口：資產頁點選信用卡帳戶 → **對帳**分頁

### 兩種對帳方式

**方式一：上傳帳單（AI 解析）**
1. 上傳信用卡帳單 PDF 或截圖。
2. 系統傳送至 Gemini AI 解析，30 秒內回傳消費明細（日期、商家、金額）。
3. 系統自動比對 App 內現有交易：
   - 金額相符且日期差 ≤ 3 天 → 自動勾選（標記「日期偏差 N 天」）
   - 完全找不到對應記錄 → 標記為「缺少記錄」，可直接新增交易
4. 確認所有項目後點「完成對帳」。

**方式二：手動對帳**
1. 點選「手動對帳」，系統列出本期所有信用卡消費。
2. 逐筆勾選確認。
3. 底部進度條即時顯示「已確認 X / N 筆」及已確認金額。
4. 若有未勾選項目，點完成時會跳警示，需二次確認。

### 對帳完成後

- 若有現金回饋，自動從帳單金額扣除
- 建立一筆**待扣款記錄**（含淨額、截止日、來源銀行帳戶）
- 到期日當天自動執行扣款，從銀行帳戶建立現金流出交易

## 6. 信用卡回饋

入口：資產頁點選信用卡帳戶 → **回饋規則**

### 新增回饋規則

每張信用卡可設定多條回饋規則，每條規則包含：

| 欄位 | 說明 |
|------|------|
| 回饋分類或商家 | 分類名稱或關鍵字（必填） |
| 備註關鍵字 | 選填，更精確匹配商家 |
| 最低消費門檻 | 選填，單筆消費未達此金額不計回饋 |
| 回饋比例 | 百分比，如輸入 3 代表 3% |
| 回饋類型 | 現金回饋抵扣 / 點數累積 / 存入指定帳戶 |
| 每月上限 | 選填，當月累積達上限後回饋歸零 |

### 三種回饋類型說明

- **現金回饋抵扣**：累積 TWD，對帳時自動從帳單扣除
- **點數累積**：累積點數，可設定每點 = X TWD 換算顯示參考值
- **存入指定帳戶**：累積 TWD，手動確認收到後，系統建立一筆收入交易至指定帳戶並重置待收金額

### 每筆交易回饋預覽

使用信用卡記帳時，系統即時顯示預估回饋金額。若當月已達上限，顯示「上限已達」。

## 7. 定期交易

入口：「更多」分頁 → **定期交易設定**

### 新增定期交易範本

| 欄位 | 說明 |
|------|------|
| 金額 | 必填 |
| 類別 | 必填 |
| 帳戶 | 必填 |
| 頻率 | 每日 / 每週 / 每月 / 每年 |
| 開始日期 | 必填 |
| 結束日期 | 選填 |
| 專案 | 選填 |
| 備註 | 選填 |
| 類型 | 支出 / 幫人付 / 投資入金 |

### 自動執行

每次開啟 App 時，系統自動檢查是否有到期未產生的定期交易，並自動建立對應記錄（帳戶餘額即時更新）。若錯過多期，會一次補齊並通知。

### 管理定期範本

- 編輯範本（修改金額、類別等）→ 只影響**未來**的記錄，過去已產生的不受影響
- 取消範本 → 停止產生未來記錄，過去已產生的記錄保留
- 已完成/已取消的範本在獨立的「封存」區塊查看

## 8. 帳本（個人 & 共享）

### 個人帳本

每位使用者自動擁有一本「個人」帳本，無法刪除。記帳時預設歸屬個人帳本。

### 建立共享帳本

1. 進入「帳本」分頁，點選「+」新增帳本。
2. 填入帳本名稱、選填的開始/結束日期。
3. 建立者自動成為成員。

### 邀請好友加入

1. 進入共享帳本 → **成員管理**。
2. 從已加為好友的清單中選擇對象，發送邀請。
3. 對方在帳本列表看到「待接受邀請」，可接受加入。

### 成員操作

- **建立者**：可邀請好友、可刪除帳本（連同所有成員記錄一起刪除）
- **一般成員**：可退出帳本（建立者不可退出，只能刪除）

### 交易與帳本的關係

記帳時可選擇帳本，選共享帳本的交易只出現在該帳本的交易列表中，不出現在個人帳本。

## 9. 報表

入口：「報表」分頁

### 時間區間選擇

內建快捷：本週、本月、本季、本年、上月、去年，或自訂任意起訖日期。

### 支出摘要

- 所選期間的總支出
- 按父類別排列的支出佔比（含百分比與比例條）
- 與前一個等長期間的比較

### 類別下鑽

點擊任一類別，查看該類別的子類別細項與個別交易清單。

### 趨勢圖

- 期間 ≤ 3 個月：每週一條柱狀
- 期間 > 3 個月：每月一條柱狀
- 同時顯示前 5 大支出類別與現金流（收入 - 支出）

### 帳戶餘額歷史

選定帳戶與時間範圍（1 個月 / 3 個月 / 6 個月 / 1 年），查看餘額折線圖。

## 10. 好友與邀請碼

### 你的邀請碼

每位使用者自動分配一組 **6 碼英數邀請碼**（如 `AB3X7Z`）。在「更多」頁面的帳號設定區可看到你的邀請碼，點選即複製到剪貼簿。

### 加好友

1. 進入「更多」→「好友」。
2. 在邀請碼輸入框輸入對方的 6 碼邀請碼（自動轉大寫）。
3. 輸滿 6 碼後「送出邀請」按鈕解鎖，點選送出好友邀請。
4. 不能輸入自己的邀請碼。

## 11. 帳號設定

### 匿名帳號

App 首次啟動**無需任何登入手續**，自動建立匿名帳號，直接進入主畫面。Session 持久化儲存，下次開啟無需重新登入。

### 綁定 Apple ID

在「更多」→ 帳號設定，點選**綁定 Apple ID**。完成 Apple 驗證後，帳號從訪客升級為 Apple ID 帳號，User ID 不變、資料完整保留，日後換裝置可恢復資料。

| 狀態 | 顯示 |
|------|------|
| 未綁定 | 「訪客帳號」＋「綁定 Apple ID」按鈕 |
| 已綁定 | 「已綁定 Apple ID」（按鈕消失） |
